Instructions

Welcome to Hyrule Hunt.
Your objective for this game it to find and defeat 2 enemies
which will unlock the door to the boss.
If you successfully do this you win.
There is a heart vessel in the level which you can collect giving you 2 potions,
these can be used during fights to heal so use them when you must.
Control the player by using the arrow keys and use the mouse to click on attacks in fights.
The first 2 attacks are guaranteed hits but the special attack has a 1 in 3 chance of hitting.
